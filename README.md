## Environment

- Tested on IDA Pro 7.7
- Require >= Python 3.8

## Features

- FSB, Command Injection, Path Traversal detection
- Provides Advanced GUI

## How To Use?

Open IDA script window (shift+f2) (mode: python), import and run, you can drag and drop.
